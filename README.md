AI-Powered Food Complaint Intelligence System

Overview

The AI-Powered Food Complaint Intelligence System is an intelligent workflow built using **n8n** and **Google Gemini** to automate food complaint analysis.

Instead of manually reviewing customer complaints, the system uses AI to classify complaints, assess severity, identify possible root causes, recommend responsible departments, suggest corrective and preventive actions, store the results in Google Sheets, and automatically notify stakeholders when high-risk complaints are detected.

---

 Features

* AI-powered complaint categorization
* Severity assessment (Low, Medium, High, Critical)
* Root cause identification
* Department recommendation
* Corrective action suggestion
* Preventive action recommendation
* Automatic logging into Google Sheets
* Automatic email alerts for High/Critical complaints

---
